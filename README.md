# 小宇宙桌面版

小宇宙桌面版是一个用于演示技术栈的桌面应用程序，使用 Tauri(v2) 和 Vite 构建，集成了 React、Tailwind CSS、shadcn/ui 以及 Radix UI 等前端技术栈。

## 功能特性

- 使用 Tauri 构建桌面应用，具备轻量级、高性能、跨平台的优势。
- 集成 Radix UI 组件库，提供无障碍的 UI 组件。
- 使用 Tailwind CSS 进行样式管理，配合 Tailwind Merge 提供简洁的样式合并。
- 使用 Zustand 进行状态管理，提供简洁的全局状态管理方案。
- 集成 Redux DevTools 扩展，方便开发过程中进行调试。
